# Today in History - May 14

## Event 1
📅 Year: 1918  
📝 Event: Cape Town Mayor, Sir Harry Hands, inaugurates the Two-minute silence.

## Event 2
📅 Year: 1951  
📝 Event: Trains run on the Talyllyn Railway in Wales for the first time since preservation, making it the first railway in the world to be operated by volunteers.

## Event 3
📅 Year: 1953  
📝 Event: Approximately 7,100 brewery workers in Milwaukee perform a walkout, marking the start of the 1953 Milwaukee brewery strike.
